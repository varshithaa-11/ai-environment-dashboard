# 🌍 AI Environmental & Pollution Dashboard

A real-time AI-powered dashboard that monitors weather conditions, 
air quality, and pollution levels with intelligent health recommendations.

## 🚀 Live Demo
👉 [Click here to view the app](https://ai-environment-dashboard.streamlit.app)

## ✨ Features
- Real-time weather data (Temperature, Humidity, Wind Speed)
- Air Quality Index (AQI) monitoring
- Pollution parameters (PM2.5, PM10, NO2, CO, O3, SO2)
- AQI Health Risk Gauge
- City AQI Comparison Chart
- AI-powered health tips using Groq AI

## 🛠️ Tech Stack
- Python
- Streamlit
- OpenWeatherMap API
- Groq AI (LLaMA 3.3)
- Plotly
